## User Journey Example

```
👤 User Story: Sharing a Report Template

1. 📝 User creates custom report
   "Q4 Performance Dashboard"
   Selects 12 fields (task ID, title, status, etc.)
   
2. 💾 Template saved successfully
   Appears in "Saved Templates" grid
   
3. 🖱️ User hovers over template card
   Blue "Share" button fades in smoothly
   
4. 🔗 User clicks "Share"
   ShareModal opens with beautiful animation
   
5. ⏰ User selects "1 month" expiration
   Calendar icon highlights the option
   
6. ✨ User clicks "Generate Link"
   Loading state shows briefly
   Share link created instantly
   
7. 📋 User clicks copy button
   ✓ "Copied!" confirmation appears
   Link in clipboard
   
8. 📧 User shares via email
   Recipient receives link
   
9. 🌐 Recipient opens link
   ┌───────────────────────────┐
   │ 🎨 Beautiful gradient BG  │
   │ 📊 Bouncing report icon   │

   └───────────────────────────┘
